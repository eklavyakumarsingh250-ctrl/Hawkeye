🛠️ Advanced Toolbox (run locally)

🔍 Subfinder

Fast subdomain enumeration

subfinder -d example.com -o subdomains.txt

⚡ HTTPX

Probe for live hosts

httpx -l subdomains.txt -o live.txt

🧬 Nuclei

Template‑based vulnerability scanner

nuclei -l live.txt -t ~/nuclei-templates/ -o vulns.txt

🗡️ Katana

Fast crawler (URL discovery)

katana -u https://example.com -o urls.txt

💉 SQLmap

Automated SQL injection detection

sqlmap -u "https://example.com/page?id=1" --batch

🔐 Hydra

Brute‑force login forms / services

hydra -l admin -P wordlist.txt example.com http-post-form "/login:user=^USER^&pass=^PASS^:F=incorrect"

🗺️ Nmap

Port scanning & service detection

nmap -sV -sC -T4 example.com

🔥 DDoS (educational only)

⚠️ Use only on your own infrastructure

hping3 -S --flood -V -p 80 example.com
⚠️ This tool is for ethical testing only. Use only on targets you own or have explicit permission.
All scans are performed client‑side; no data is sent to any server except the target itself.
